Skills

HTML5

HTML (HyperText Markup Language) is the standard language used to create web pages. It structures content using elements like headings, paragraphs, images, and links. HTML uses tags to organize and display information in a browser. It works together with CSS for styling and JavaScript for interactivity. HTML is the basic foundation of every website.

CSS3

CSS is a style sheet language used to design and style web pages. CSS3 is the modern version of CSS that adds advanced features like animations, transitions, gradients, and flexible layouts. It controls colors, fonts, spacing, and positioning of elements on a webpage. CSS3 works together with HTML to create visually appealing websites. It helps developers make responsive and attractive user interfaces.

JavaScript

JavaScript is a programming language used to make web pages interactive. It allows developers to add dynamic features like buttons, forms, animations, and pop-up windows. JavaScript works together with HTML and CSS to build modern websites. It can respond to user actions such as clicks, typing, or scrolling. JavaScript is one of the most important languages in web development.

UI/UX

UI/UX Design refers to the design of a product’s interface and user experience. UI (User Interface) focuses on the visual elements of a website or app, such as buttons, colors, and layouts. UX (User Experience) focuses on how easy and comfortable it is for users to interact with the product. Good UI/UX design helps users navigate a website smoothly and find information quickly. It is important for creating attractive and user-friendly digital products.

Git & Github

Git is a version control system used to track changes in code during development. It allows developers to save different versions of a project and collaborate with others. GitHub is an online platform that hosts Git repositories and helps developers share and manage their code. Developers use Git to commit and track changes, and GitHub to store and collaborate on projects. Together, they are essential tools for modern software development.
